Here are some of my various Audio projects I have worked on, primarily utilizing VCV rack 2 to create generative music, which is music that, through electronics, is able to utilize past musical detail to impact the current sound. In essence, all of the initial data changes every aspect of the song, and changing anything slightly will create a whole new set of sounds.

Project Name Description Video File
Generative Thumps This audio file/video is a vcv rack patch where I have a series of different thumping instruments, and I can change the tone/frequency of the thumps by adjusting various dials in the patch
Random Clocks This piece of generative music is created using random clocks, allowing it to generate diffferent beats through various cycles
Waves This piece of generative music is created using wave forms, which osciliate over the course of the patch to create different sounds
